            How the Engine Works
            
      
            A 7-stage pipeline from query to structured visual explanation.
            

      
            
        
            01
            🔍
            Web Discovery
            DuckDuckGo finds the most relevant pages for your query.
            
        
            →
            
        
            02
            🌐
            Scraping
            Playwright + Trafilatura extract clean text from webpages.
            
        
            →
            
        
            03
            📚
            RAG Retrieval
            Sentence-Transformers + FAISS find the most relevant chunks.
            
        
            →
            
        
            04
            🤖
            LLM Synthesis
            LLaMA 3.3 70B via Groq generates a structured explanation + D2 diagram code.
            
        
            →
            
        
            05
            📊
            Kroki Diagrams
            D2 code is sent to Kroki.io → returns a rendered SVG architecture diagram.
            
        
            →
            
        
            06
            🖼️
            Wikipedia + YouTube
            A relevant research image and tutorial video are fetched automatically.
            
        
            →
            
        
            07
            📖
            Structured Output
            Beginner → Advanced breakdown with diagram, code and cited sources.
